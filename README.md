# Waste Classification using SVM

## Overview

This project is **Classifying Waste Material from Images using SVM** that classifies waste images as **Organic** or **Recyclable** using an SVM model and a Streamlit interface.

## Tech Stack

* Python
* Streamlit
* OpenCV
* Scikit-image (HOG)
* NumPy
* Scikit-learn

## How It Works

1. Upload an image
2. HOG + color histogram features are extracted
3. SVM model predicts the waste category
